More on regular expressions and Python...

For more information on regular expressions, check out wikipedia
	(http://en.wikipedia.org/wiki/Regular_expression)
	and the official Python documentation:
	http://docs.python.org/library/re.html
	
There are several useful Python regular expression "cheatsheets", one is at Debuggex:
	https://www.debuggex.com/cheatsheet/regex/python
and another is a nice pdf from Johnathon Hartley's gitHub:	
	https://cloud.github.com/downloads/tartley/python-regex-cheatsheet/cheatsheet.pdf
